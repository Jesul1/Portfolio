Seleste

Seleste was a school project and the very first game I created in Unity. The objective of the project was to replicate the mechanics of the 2D platformer Celeste.

I thoroughly enjoyed working on my first game and got carried away, dedicating my free time and weekends to expanding it beyond the initial scope. I added animations, collectibles, moving platforms, checkpoints, and many other features to bring the game to life.

Every aspect of the game—art, mechanics, and design—was created entirely by me. This project sparked my passion for game development and taught me the fundamentals of Unity and 2D game design.

Tools used:
- Unity 2019
- Visual Studio 2019

-PhotoShop
