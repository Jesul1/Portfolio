VR survival game

After makeing the sandbox mobile game with Joni we set our sights on making a VR game. We tried to come up with ideas on what to do in vr and what we hadn't seen before.

we came up with a procedurally generated survival game where everything would be done by hand the plan was to avoid adding any kind of ui to the game so it would feel really imersive without any "gamey" elements, like building would be handeled by nailing planks, smelting by actually placeing items in ovens and crafting by attaching items in anyway you pleased.

however this being our 2nd 3D game ever and our first VR game ever with procedural generation, witch is something we hadn't done ever before, it was a bit too much for us to handle, but we learned a lot making it.

i personally learned alot about procedural generation, shaders in unity, VR development and not to set the scope of a game so large for a small team. it's a shame we didn't have time to finish it. i'd really like to revisit the idea some day.
