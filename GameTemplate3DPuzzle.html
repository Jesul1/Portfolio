3D Puzzle game

This was my first-ever 3D game and our first group project, where we learned to develop games as a team and use version control tools like Git Extensions. Inspired by Portal, we set out to create a 3D puzzle game.

The game was set in ancient Egypt, where the player, trapped in a pyramid, must escape by solving puzzles centered around toggling blocks on and off and freezing them in time.

I programmed most of the mechanics, including player movement, puzzle interactions, and the time-freezing mechanic. I also handled all the level design and graphics.
