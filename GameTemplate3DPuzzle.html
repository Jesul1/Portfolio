3D Puzzle game

This project was my first ever 3D game and our first group project, so we had to learn how to develop games in teams and how to use version control tools like git extentions. Our team wanted to try and create a 3D puzzle game with inspiration from portal.

The game was to be set in anchient egypt where the player is trapped in a pyramid and has to escape. The puzzles would revolve around a mechanic where the player could toggle blocks on and off and also freeze them in time.

i programmed most of the mechanics in this game like, player movement, puzzle mechanics and the time freezing mechanic. I also did all the level desing and graphics.

With very little involment from my team members the project failed to see completion, but i learned alot from it.

Tools used:
- Unity 2019
- Visual Studio 2019
